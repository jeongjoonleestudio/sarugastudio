## Home info box

The yellow homepage info box reads from the Google Sheet URL in `projects.js`.
Create a published sheet tab named `home_info` with a `content` column. The tab
must be included in the spreadsheet's "Publish to web" settings. Edit the text in
that column to update the yellow box without changing the code. The current
published gid for that tab is `1332433738`.
